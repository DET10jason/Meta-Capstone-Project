🍋 Little Lemon Restaurant Landing Page
A responsive, high-performance landing page for the "Little Lemon" restaurant, featuring a full-cycle table reservation system. This project showcases the use of React, React Router, and component-driven development.

✨ Features
Table Reservation System: 
  A dedicated booking flow that takes users from the landing page to a multi-step reservation form.
Dynamic Menu Section: 
  Displays weekly specials with custom-styled cards.
Customer Testimonials: 
  A social proof section featuring a clean, grid-based layout for reviews.
About Section: 
  A narrative component detailing the history of the restaurant with responsive image positioning.
Global Navigation: 
  A sticky/fixed header and comprehensive footer for easy site navigation.
  
🛠️ Tech Stack
Core: React.js
Navigation: React Router (for seamless page transitions)
Styling: CSS3 / Modules (customized for the Little Lemon brand palette)
Icons: FontAwesome / Custom SVG
